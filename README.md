# Sales Data Analysis using Excel and SQL
## Notes
This project was created as a hands-on practice to improve my skills in Excel-based data analysis and basic SQL.

## Overview
This project analyzes a sales dataset to identify trends, top-performing products, and region-wise performance using Excel and SQL.

## Tools Used
- Microsoft Excel
- SQL

## Key Tasks
- Cleaned raw sales data in Excel
- Created pivot tables and charts for analysis
- Used SQL queries for aggregation and reporting

## Insights
- Identified top-selling products
- Analyzed region-wise sales distribution
